# 🛒 E-Commerce Website

A modern and responsive E-Commerce website developed using HTML, CSS, and JavaScript. The project provides a complete front-end shopping experience with multiple pages, interactive features, and a clean user interface.

---

## 🚀 Features

- Responsive Design
- Home Page
- Product Listing
- Product Details
- Shopping Cart
- Wishlist
- Checkout Page
- User Login & Signup
- About & Contact Pages
- Order Success Page
- Interactive UI using JavaScript

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎯 Key Functionalities

- Browse products

- Checkout workflow
- User authentication pages
- Responsive navigation
- Interactive shopping experience
